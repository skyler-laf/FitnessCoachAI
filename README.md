# 🏋️ Fitness Coach AI

A full-stack web application that uses the OpenAI API to generate personalized one-week workout plans based on a user's fitness profile.

## Live Demo

Frontend (Vercel):
https://fitness-coach-ai-green.vercel.app

Backend (Render):
https://fitnesscoachai.onrender.com

https://github.com/skyler-laf/FitnessCoachAI

## Features

- Personalized AI-generated workout plans
- User-friendly React interface
- Express.js backend
- OpenAI API integration
- Weekly workout schedule
- Daily workouts
- Nutrition tips
- Motivational message
- Responsive design

## How It Works

1. Enter your fitness information.
2. Click **Generate Workout Plan**.
3. The frontend sends your information to the Express backend.
4. The backend calls the OpenAI API.
5. The AI returns a personalized workout plan.
6. The workout plan is displayed on the webpage.
